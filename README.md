# TesraBot

**Packages REQUIRED**

`discord.js` (npm install discord.js)

`ms` (npm install ms)

`os` (npm install os)

**Setting up the BOT**

Fill in your details in:
Credentials (`settings/credentials.json`) **<RENAME FILE TO credentials.json>**
```js
  {
  "botID": "BOT ID",
  "ownerID": "OWNER ID",
  "botToken": "BOT TOKEN"
}
```
Fill in your details in: 
Config (`settings/config.example.json`) **<RENAME FILE TO config.json>**
